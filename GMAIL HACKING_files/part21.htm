List of Gmail Shortcuts


Shortcut Key

Definition

Action

C

Compose

Compose a new email.

/

Search

Puts your cursor in the search box.

K

Move to newer conversation

Move cursor to newer conversation.

J

Move to older conversation

Move cursor to older conversation.

N

Next message

Next message in conversation.

P

Previous message

Previous message in conversation.

O or <Enter>

Open

Expand / collapse a message

U

Return to conversation list

Return to conversation view from message view.

X

Select conversation

Automatically checks and selects a conversation so that you can archive, apply a label or choose an action from the dropdown menu to apply to that conversation

S

Star a message

Star the conversation.

!

Report spam

Mark as spam and move to Spam View.

R

Reply

Reply to an email.

A

Reply all

Reply to all message

F

Forward

Forward the current email

<Esc>

Escape from input field

Removes the cursor from your current input field.


Backup your Gmail Inbox Online

Create a new Gmail account and under Settings –> Accounts and Import –> Check mail using POP3 –> Add POP3 email account; enter the email address of your main Gmail account that you want to backup.
